# Media Assets

Store all reusable site media here.

## Folders

- `images/` - covers, author photos, blog visuals, UI art
- `video/` - trailers, teasers, promo clips
- `audio/` - narration clips, soundtrack samples
- `documents/` - downloadable PDFs or media kits

## Usage in Next.js

Files under `public/` are served from the site root.

Example:

- File path: `public/media/images/book-cover.jpg`
- URL path: `/media/images/book-cover.jpg`
